# 📖 About Repository

The **Java Collections Framework** provides powerful data structures and algorithms that simplify data storage, retrieval, manipulation, and organization.

This repository contains **six industry-oriented console-based mini projects** that strengthen problem-solving skills while demonstrating the practical usage of Java Collections.

Each project focuses on solving a real-world scenario using an appropriate collection type.

---

# 🚀 Mini Projects Overview

## 1️⃣ Manage Employee Details

### Description

A console-based Employee Management application that stores employee details and displays them sorted by **First Name**.

### Collections Used

- TreeSet
- Comparator
- Generics

### Concepts Covered

- Object Sorting
- Custom Comparator
- TreeSet
- Encapsulation

---

## 2️⃣ String Operations

### Description

A menu-driven application that performs various string operations such as insertion, searching, deletion, and display.

### Collections Used

- ArrayList

### Concepts Covered

- CRUD Operations
- ArrayList
- Menu Driven Programming

---

## 3️⃣ Collect Unique Symbols From Set of Cards

### Description

Collects playing cards until all four unique symbols are gathered and displays the first occurrence of each symbol in alphabetical order.

### Collections Used

- TreeSet
- HashSet

### Concepts Covered

- Unique Elements
- Set Operations
- Object Comparison

---

## 4️⃣ Collect and Group Cards

### Description

Groups playing cards based on their symbols and displays the grouped cards along with the total count and sum of card values.

### Collections Used

- TreeMap
- ArrayList

### Concepts Covered

- Grouping Data
- Maps
- Collections Framework

---

## 5️⃣ Set of Boxes

### Description

Stores only unique boxes based on their volume using custom implementations of `equals()` and `hashCode()`.

### Collections Used

- HashSet

### Concepts Covered

- equals()
- hashCode()
- Object Equality
- Hash-Based Collections

---

## 6️⃣ String Operations with ArrayList

### Description

Performs five advanced string manipulation operations and returns all outputs using an `ArrayList<String>`.

### Collections Used

- ArrayList

### Concepts Covered

- String Manipulation
- Collections
- ArrayList Operations
