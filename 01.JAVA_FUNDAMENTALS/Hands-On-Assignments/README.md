## 📖 About

This directory contains my solutions to the **Hands-On Assignments** from the **Wipro TalentNext – Java Fundamentals** training program.

The assignments are designed to strengthen programming fundamentals by providing practical exercises on Java syntax, control structures, arrays, object-oriented programming, and problem-solving techniques.

All programs are implemented in Java following clean coding practices and are organized topic-wise for easy navigation.

## 🚀 How to Run

Compile a Java program:

```bash
javac FileName.java
```

Run the program:

```bash
java FileName
```

For programs using command-line arguments:

```bash
java FileName arg1 arg2
```
